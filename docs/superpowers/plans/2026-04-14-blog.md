# Blog Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `/blog` index page with a 3-column card grid and 5 individual post pages (`/blog/[slug]`) with a full article layout.

**Architecture:** Static TypeScript data file (`src/lib/blog.ts`) exports a `POSTS` array — no CMS. The blog index reuses `HeroSection` with a new optional `centred` prop. Individual post pages render a full-width featured image banner (replacing the hero), then an article body with intro, sections, conclusion, and a `CtaBanner`. Posts are statically generated via `generateStaticParams`.

**Tech Stack:** Next.js 14 App Router, TypeScript, Tailwind CSS, `next/image`, `next/link`, `next/navigation` (`notFound`).
